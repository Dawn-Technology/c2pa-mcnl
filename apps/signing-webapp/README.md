# signing-webapp
This is a web application for signing media files with C2PA manifests. It allows users to upload media files, create C2PA manifests, and sign the media with the manifest information to ensure authenticity and provenance.

### Features
- Upload media files and C2PA manifests for signing.
- Upload certificates and keys to sign the media files with the manifest information.
- Create VCs (Verifiable Credentials) for digital identity verification using Decentralized Identifiers (DIDs).
- Build and sign C2PA manifests to ensure the authenticity and provenance of media files.

### Development

To run the development server for the verification web application, use the following command:

```bash
  nx serve signing-webapp
```
