# verify-webapp-validate-feature-detail

> This library was generated with [Nx](https://nx.dev).

The validation feature which will read the C2PA manifest and show the details of the manifest in the UI. This will be used in the webapp to show the details of the manifest to the user.

## Running unit tests

Run `nx test verify-webapp-validate-feature-detail` to execute the unit tests.

## Running linter

Run `nx lint verify-webapp-validate-feature-detail` to execute the lint via [ESLint](https://eslint.org/).
